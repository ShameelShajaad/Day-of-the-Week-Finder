## ğŸ’¡ Description
The **Day of the Week Finder** is a **web-based application** that calculates the day of the week for any given date.  
Users can enter a **year, month, and day**, and the app will display the corresponding day (Sunday, Monday, etc.) dynamically.  

This project demonstrates **front-end web development skills**, including **form input handling, validation, DOM manipulation, and event-driven programming** using **JavaScript**.

---

## âš¡ Features

1. **Date Input**
   - Users can enter **year (YYYY), month (MM), and day (DD)**
   - Input fields are validated for correct ranges and leap years

2. **Dynamic Calculation**
   - Calculates the day of the week using a **mathematical formula**
   - Supports **leap years** and checks **month-specific day limits**

3. **Error Handling**
   - Alerts for invalid input, such as:
     - Missing fields
     - Invalid year
     - Invalid month or day
     - February constraints for leap and non-leap years
     - Months with 30 days only

4. **Interactive Output**
   - Displays the result dynamically on the page
   - Uses a styled result box for clear user feedback
